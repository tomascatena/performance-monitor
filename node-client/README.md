# Node Client

It will be a node client that will be installed on the client machine and constantly send performance information to the server.

The following information will be sent to the server:

1. CPU load (current)
2. Memory usage:
   - Total memory
   - Free memory
3. OS information:
   - OS type
   - OS version
   - OS release
4. Machine uptime
5. CPU information:
   - Type
   - Number of cores
   - Clock speed
